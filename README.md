- 회사 홈페이지
- 당사의 기본 정보 전달, 웹을 통한 고객소통 창구 마련이 목적

### 구성 및 기능
- 메뉴 구분: 홈 / 핵심역량 / 사업분야 / 고객지원 / 회사소개
- <고객지원>에서는 제출양식 포함, 제출 시 내용을 메일로 전송되도록 함
- 이 외 페이지는 텍스트, 이미지, 디자인 요소를 배치 (백엔드 불필요)
- 반응형 UI 적용

### 기술 스택
- Flutter
- Dart
- Hosting (결정 필요, Firebase 고려 중)
